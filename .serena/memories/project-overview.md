Project: Kitchen Odyssey (branded as "CookHub" in UI). React 19 + Vite + Tailwind v4. Client-only SPA with localStorage persistence (keys prefixed cookhub_). HashRouter. Roles: admin, user. User statuses: active, inactive, pending, suspended. Core features: auth (login/signup/logout), recipe CRUD with admin approval workflow, search/filter/sort, star ratings/reviews, likes/favorites, profile viewing/editing, admin dashboards (stats/users/recipes), daily stats tracking, activity logs. Guest mode is PLANNED but NOT yet implemented. No backend API.
